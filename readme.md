# Productos Innovadores

Proyecto web desarrollado con HTML5 y CSS3 para la presentación de productos mediante una interfaz simple, accesible y optimizada.

## Descripción

Este sitio web muestra una galería de productos utilizando una estructura semántica HTML, estilos personalizados con CSS y optimización de imágenes mediante formato WebP.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub
- Imágenes optimizadas en formato WebP

## Características implementadas

- Estructura HTML semántica utilizando etiquetas como:
  - header
  - nav
  - main
  - section
  - article
  - footer

- Diseño responsive adaptable a dispositivos móviles.

- Optimización de imágenes utilizando:
  - Formato WebP.
  - Atributo alt para accesibilidad.
  - loading="lazy" para mejorar la carga.
